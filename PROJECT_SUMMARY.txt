0. PROJECT OVERVIEW
================================================================================

TradeX is a browser-only paper trading platform. Users get a virtual $10,000
wallet on signup, trade 20 default US stocks (100 in data file) at live Finnhub
prices, compete on a leaderboard, and set price alerts with optional email.
Admins manage users, fund requests, and audit logs via Supabase RPCs.

Architecture:

  Browser (React SPA)
    ├── Supabase Auth + Postgres (RLS) — users, wallets, trades, watchlist, alerts
    ├── Finnhub REST — quotes, news (queued + cache)
    ├── Finnhub WebSocket — Dashboard, Portfolio, Watchlist (throttled UI)
    ├── Finnhub CDN — stock logos
    └── TradingView CDN — charts, tape, widgets

Documentation:
  README.md                      — Quick start, deploy
  supabase/SUPABASE_SETUP.md     — Supabase setup (SQL, Auth, emails)
  supabase/ALERTS_SUPABASE_SETUP.md — Price alerts + Resend email

6. P&L & METRICS (IMPORTANT)
================================================================================

  Unrealized P&L (Portfolio) = market value of holdings − cost basis
  Realized P&L (Admin)     = sum of (sell proceeds − cost) on SELL rows only
  Total return %           = (cash + holdings) vs wallets.total_deposited

  Realized $0 with many buys = normal if few/no sells
